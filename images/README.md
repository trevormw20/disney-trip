# Images folder

Drop gear example images here so the survey shows real products instead of CSS mockups.

## Expected filenames

- `keychain-example.png` — Stitch-style embroidered keychain loops image
- `lanyard-example.png` — Toy Story-style full neck lanyard image

`.png`, `.jpg`, or `.webp` all work — just match the filename before the extension.

If an image is missing, the survey automatically falls back to the CSS mockup, so it never breaks.

## How to save

1. Right-click each image in your chat → Save image as…
2. Save to this folder with the exact filename above
3. `git add images/ && git commit -m "add gear example images" && git push`
4. Open the survey — the real images now show
